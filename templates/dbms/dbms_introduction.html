Introduction to Database Management Systems (D.B.M.S.)

1. What is D.B.M.S.?

A Database Management System (DBMS) is software used to store, manage, and retrieve data efficiently. It organizes data in a way that it can be easily accessed, manipulated, and updated.

Types of Databases

  • Relational Databases: Data is stored in tables (e.g., MySQL, PostgreSQL). Relationships between tables are defined using keys.
  • NoSQL Databases: Handles unstructured data (e.g., MongoDB) and is more flexible in dealing with large volumes of data.

2. DBMS vs File Systems

DBMS provides features like reduced data redundancy, data consistency, and enhanced security compared to traditional file systems.

Database Architecture

  • 1-Tier: All tasks handled on a single machine (centralized).
  • 2-Tier: Client-server architecture.
  • 3-Tier: Application layer added between client and server.

3. SQL and Basic Operations

SQL: Structured Query Language is used to interact with databases. It includes:

  • DDL (Data Definition Language): For defining the structure (CREATE, ALTER).
  • DML (Data Manipulation Language): For data manipulation (SELECT, INSERT, UPDATE, DELETE).
